# SpaceN - Space Tourism Landing Page 🚀🌌

A futuristic landing page for a fictional space tourism company, built as my first hands-on project to master the core fundamentals of web development.

## 🛠 Technical Breakdown
In this project, I focused on mastering the core fundamentals of web layout and styling without using modern frameworks.

* **Layout Architecture:** Structured using semantic HTML and `div` containers, managing element dimensions precisely.
* **Positioning & Floats:** Utilized `float` for alignment and `absolute positioning` for text overlays on background images.
* **Transparency & Design:** Implemented manual "Glassmorphism" effects using `RGBA` color values and `opacity`.
* **Interactive Hover Effects:** Added CSS `transitions` for smooth user interactions on buttons.
* **Asset Management:** Managed external Google Fonts, icons, and local images using relative paths.
